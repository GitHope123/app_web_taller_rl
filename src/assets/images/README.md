# Carpeta de Imágenes

Esta carpeta contiene los recursos visuales de la aplicación.

## Estructura Recomendada:

```
images/
├── logos/
│   ├── logo-full.svg          # Logo completo (para drawer expandido)
│   ├── logo-icon.svg          # Isotipo/icono (para drawer colapsado)
│   ├── logo-full-dark.svg     # Logo completo modo oscuro (opcional)
│   └── logo-icon-dark.svg     # Isotipo modo oscuro (opcional)
├── icons/
│   └── [iconos personalizados]
└── backgrounds/
    └── [imágenes de fondo]
```

## Especificaciones de Logos:

### Logo Completo (Drawer Expandido)
- **Dimensiones**: 180-200px × 40-50px
- **Formato**: SVG o PNG transparente
- **Uso**: Cuando el drawer está abierto (240px de ancho)

### Logo Icono (Drawer Colapsado)
- **Dimensiones**: 40-48px × 40-48px
- **Formato**: SVG o PNG transparente
- **Uso**: Cuando el drawer está cerrado (65px de ancho)

## Formatos Soportados:
- SVG (recomendado para escalabilidad)
- PNG (con transparencia)
- JPG (para fotografías)
- WebP (para optimización)

## Notas:
- Usa resolución 2x o 3x para pantallas de alta densidad
- Optimiza las imágenes antes de subirlas
- Considera versiones para modo claro y oscuro
